vooki

全自动web漏洞扫描器，提供以下不同类型的扫描

  • 全扫描
  • 基本扫描
  • 手动扫描
  • 履带式
  • 域和主机扫描器
  • 密码学

1.全扫描

完全扫描检查从浏览器收集的所有 URL。它包括 web-spidering 和 CSRF 令牌绕过。全扫描检查目标 Web 应用程序的所有页面，检测复杂的登录机制并扫描授权背后的页面是否存在漏洞。这是我们强烈建议您在您的网站上执行的扫描。

2.基本扫描

基本扫描适用于没有身份验证的网站和静态网站。默认情况下，此基本扫描包括一个爬虫。所以基本扫描首先抓取，从网站收集 URL，开始扫描并最终提供最可靠的结果。

3.手动扫描

手动扫描由 HTTP(S) 拦截器和 HTTP 请求编写器组成。您可以编辑 HTTP 请求，删除它并将其发送到服务器。Vooki 中的这一独特设施将帮助您执行高效的渗透测试。

4. 履带式

Vooki crawler 对您的网站进行深入扫描，并显示域中可用的网页列表。收集的 URL 将在捕获的 URL 选项卡中可用。

5.域和主机扫描器

域和主机扫描器会扫描您的整个网站，并获取有关开放端口、服务器、DNS 信息、Web 存档、服务器地理位置和“Whois”数据的各种重要信息。

6.密码学

密码学部分由以下模块组成：

  • 编码器/解码器
  • 加密
  • 散列
